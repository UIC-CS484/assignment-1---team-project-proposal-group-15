Transport Suggestion

API Information:

  • CTA Bus Tracker, Train Tracker, Chicago Crime Portal, and OpenWeather API are being used.


  • The Data received from API's will help in forming a collective desicion to take a particular route with a certain transportation or not.


  • Data collected from the user would be the start location, drop location, and the starting time of the journey.

  • API Entities Used

    OpenWeather API: coord, weather.main, main.temp, main.feelslike, wind.speed

    Crime API: date, block, latitude, longitude, location

    Bus API: dly, dstp

    Train API: ././isDly, ./././isDly, ./././arrT, ././arrT

User Information:

  • People travelling using the CTA will use this site. They need information that reflects realtime situations that may have an effect on their personal safety.


  • Our focus is to help the user in making an informed desicion while planning to travel along a particular route.

    • Goals:

    Estimate current arrival time of public transportation.

    Display delays, extreme weather, and crime warnings.
  • There is no one-stop shop that combines all the data necessary when making a transportation choice. Bus and train times on map applications are usually inaccurate. For crime information, users specifically have to search up news their area. Combining these three types of data will help users make better travel choices.
