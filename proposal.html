Transport Suggestion

RESTFul API's used:
CTA Bus Tracker,Train Tracker and Weather API

How will this api give insight to users?
The data received from API's will help in forming a collective desicion to take a particualr route or not by providing information on delays, extreme weather, and crime warnings.

What are we collecting from the users?
Start location,drop location and starting time of the journey.

Identify all entities/tables we might use:
USER -> fname, lname, gender, email, password, start_location, drop_location and starting_time
Who will potentially use this site?
People travelling using the CTA will use this site.

Site Goals?
1. Our focus is to help the users in making an informed desicion while planning to travel along a particular route.
2. They need information that reflects realtime situations that may have an effect on their personal safety.
3. Based on gathered travel data (anonymised ofcourse) we want to add in more feature to help the users.

Why did we select this topic?
We selected this project because many people including many UIC students use CTA for daily commuting.
There is no one-stop shop that combines all the data necessary when making a transportation choice. Bus and train times on map applications are usually inaccurate. For crime information, users specifically have to search up news their area. Combining these three types of data will help users make better travel choices.
