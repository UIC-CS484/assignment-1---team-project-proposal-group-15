Transport suggestor


An application which gives an informed decision to take a particular route or not

RESTFul API's used:
CTA Bus Tracker,Train Tracker and Weather API

How will this api give insight to users?
The Data received from API's will help in forming a collective desicion to take a particualr route or not

What are we collecting from the users?
Start location,drop location and starting time of the journey

Identify all entities/tables we might use:
USER -> fname, lname, gender, email, password, start_location, drop_location and starting_time
Who will potentially use this site?
People travelling using the CTA will find this useful

1. Our focus is to help the user in making an informed desicion while planning to travel along a particular route

2. Based on gathered travel data (anonymised ofcourse) we want to add in more feature to help the users


Why did we select this topic?
We selected this project because many people including most of UIC students use CTA for daily commuting, this will help them make informed decision"
